# AI Math App 🧮

一个专为学前儿童设计的数学学习游戏 PWA 应用，针对 iPad 进行了优化。

## ✨ 特性

- 🎮 **4种游戏模式**：数数、加减法、数字配对、大小比较
- 📱 **PWA 应用**：可安装到主屏幕，支持离线使用
- 🎨 **儿童友好设计**：大按钮、鲜艳色彩、趣味动画
- 📐 **iPad 优化**：完美适配横竖屏，触摸交互流畅
- 🔊 **音效反馈**：答对/答错有不同声音提示
- 💾 **进度保存**：自动保存学习进度
- 🌍 **中英文支持**：双语界面

## 🎯 游戏模式

### 1. 数数游戏 (Counting)
数一数屏幕上有多少个物品，选择正确的数字！

### 2. 简单加减法 (Arithmetic)
通过可视化物品学习加减法，让数学变得直观有趣。

### 3. 数字配对 (Matching)
将数字与对应数量的物品配对，锻炼记忆力。

### 4. 大小比较 (Comparison)
学习大于、小于、等于的概念。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

访问 http://localhost:3000 查看应用。

## 📱 安装 PWA

在 iPad/iPhone Safari 中：
1. 打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"

## 🛠️ 技术栈

- **框架**: Next.js 14 + TypeScript
- **UI**: shadcn/ui + Tailwind CSS
- **PWA**: next-pwa
- **部署**: Vercel / GitHub Pages
